Бул эки хост бир сабнетте жатуусун табуу учун алардын сабнет ID лерин таап алуу керек. IP адрестин subnet адресин аныктоо үчүн логикалык ЖАНА (AND) деп аталган биттик операцияны колдонбуз. IP адрес менен subnet mask адрестеринин биттерин логикалык ЖАНА (AND) операциясы менен колдонуп чыгабыз. Логикалык ЖАНА (AND) операциясы иштөө принциби: эгер эки бит тең 1 болсо, жообу 1 болот. Эгер биттердин бирөө 0, бирөө 1 же экөө тең 0 болгон болсо жообу 0 болот. Subnet адресин эсептейли:

host 1 учун:

  • IP address: 11000000.10101000.00001010.00110010 (192.168.10.50)
  • Subnet mask: 11111111.11111111.11100000.00000000 (255.255.224.0)
  • Network ID: 11000000.10101000.00000000.00000000 (192.168.0.0)

host 2 учун:

  • Subnet mask: 11111111.11111111.11100000.00000000 (255.255.224.0)
  • Network ID: 11000000.10101000.00000000.00000000 (192.168.0.0)

Бул хосттун(host 2) subnet адреси 192.168.0.0 болот.
